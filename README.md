# gorepair-rest-api

Final backend project alterra academy kampus merdeka.

## API Server technology stack is

- Server code: **golang**
- REST Server: **fiber v2**
- Database: **MySQL**, **MongoDB**
- ORM: **gorm v2**

## Public API used:

- Free IP Geolocation API
- Real Email

## Other technology:

- jwt
- bcrypt
- validator
- logrus for logger
- scribbleDB for local_db (save jwt refresh)
